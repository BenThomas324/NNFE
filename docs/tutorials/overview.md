
# Overview of CARDIAX-NNFE

CARDIAX-NNFE is a residual-based method for scientific machine learning where we merge traditional finite element computation with modern machine learning tools...


The structure of the code is very modular, allowing flexibility to solve problems through configuration files or getting your hands dirty and coding specific components to try out new things.

To see how the configuration files are organized, go [here](configurations.md).
